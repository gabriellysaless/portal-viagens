# 🌍 Portal Viagens

> **Projeto desenvolvido durante o módulo 22 de Next.js da EBAC e organizado em um repositório próprio para compor meu portfólio.**

Aplicação desenvolvida com **Next.js** para apresentar destinos turísticos, utilizando **rotas dinâmicas** e boas práticas de organização de projetos.

O objetivo deste projeto foi praticar conceitos do **App Router**, navegação dinâmica, componentes reutilizáveis e estilização com **CSS Modules**.

## 🚀 Tecnologias

- Next.js
- React
- TypeScript
- CSS Modules

## ✨ Funcionalidades

- Listagem de destinos
- Página de detalhes para cada destino
- Rotas dinâmicas
- Navegação entre páginas
- Componentes reutilizáveis
- Layout responsivo

## 📂 Estrutura do projeto

```text
src/
 ├── app/
 ├── components/
 ├── data/
 ├── types/
 └── styles/
```

## ⚙️ Como executar

Clone o repositório:

```bash
git clone https://github.com/gabriellysaless/portal-viagens.git
```

Entre na pasta do projeto:

```bash
cd portal-viagens
```

Instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Abra o navegador em:

```
http://localhost:3000
```
